Requirements
************

*pyreports* is written in python3 (3.6 and higher).

Here are all the external libraries necessary for the proper functioning of the library:

- `tablib <https://tablib.readthedocs.io/en/stable/>`_
- `ldap3 <https://ldap3.readthedocs.io/en/latest/>`_
- `pymssql <https://pymssql.readthedocs.io/en/latest/>`_
- `mysql-connector-python <https://dev.mysql.com/doc/connector-python/en/>`_
- `psycopg2-binary <https://www.psycopg.org/docs/index.html>`_


Installation
************

.. code-block:: console

    $ pip install --user pyreports
